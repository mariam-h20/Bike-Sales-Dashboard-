# Bike-Sales-Dashboard-
This is an Excel dashboard that analyzes bike sales data to understand customer behavior and purchasing patterns. The dashboard provides insights based on demographics, income, age, and commute distance.

Dataset

The dataset contains information on:

	•	Customer gender
	•	Marital status
	•	Region
	•	Education level
	•	Income
	•	Age bracket
	•	Commute distance
	•	Bike purchase (Yes / No)

  Tools Used

	•	Microsoft Excel
	•	Pivot Tables
	•	Pivot Charts
	•	Slicers

  Key Insights

	•	Male customers have a higher average income per purchase than female customers.
	•	Middle aged customers are more likely to purchase bikes compared to adolescents and older customers.
	•	Customers with shorter commute distances tend to purchase bikes more often.
	•	Income level and region influence bike purchase decisions.

  Dashboard Features

	•	Interactive slicers for:
	•	Marital status
	•	Region
	•	Education level
	•	Visualizations include:
	•	Average income per purchase by gender
	•	Customer age bracket analysis
	•	Commute distance vs purchase behavior
